## Pasos para instalar las dependencias del proyecto

Levantar el servidor con xamp u otro servidor que ejecute php
crear un archivo .env, copair la informacion de .env.exmaple
ejecutar los comandos
composer install
npm install
php artisan key:generate
php artisan migrate
php artisan migrate:refresh
npm run dev
php artisan serve para levantar el servidor
